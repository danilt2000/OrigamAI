   DeployPlatforms 
 Used when the application is deployed on both Microsoft SQL Server and PostgreSQL or PostgreSQL only and relevant for the Deployment Script Generator.  DataConnectionString - same as DataConnectionString in Data Connection section 
 DataService - same as DataDataService in Data Connection section 
 IsPrimary - True/False 
 Name - just a name for you
