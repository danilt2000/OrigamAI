# Reporting Features

Category: docs
URL: https://community.origam.com/t/reporting-features/2329

@system (2017-12-31 23:00:00Z):
 Reporting is integrated deep into every application. You can 
 
 Show page reports like invoices for a current record by clicking a button 
 Show list reports by showing a selection dialogue first 
 Show web reports by showing a selection dialogue and then passing the parameters to an external URL 
 
 Reports from an integrated reporting engine are shown as a ready-to-save or print PDF. 
 Reports can be localized and use built-in data layers so they do not access the database directly, leveraging built-in data models, filters, etc.
