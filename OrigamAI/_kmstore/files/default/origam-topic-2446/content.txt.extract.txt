# Workflow Call Task

Category: docs
URL: https://community.origam.com/t/workflow-call-task/2446

@system (2017-12-31 23:00:00Z):
 With this task, you can reuse workflows or split complicated workflows into simple reusable units. This task simply executes another workflow, possibly passing it some data and possibly using some data the called workflow may return.
