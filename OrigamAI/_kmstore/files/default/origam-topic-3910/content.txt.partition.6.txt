’ branch. The -u (or --set-upstream) option sets the “master” branch as the default for future git push and git pull operations. 
 The result is that your changes (files and commits) are stored on GitHub in the “master” branch, and the local ‘master’ branch is linked to the remote “origin/master” branch.
